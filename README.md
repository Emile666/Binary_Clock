# Binary Clock Project
- 20 x WS2812B leds hrs (2+4), min (3+4), sec (3+4)
- STM8S103F3 uC
- 2 x 2 LEDs for colons
- IR (TL1838) receiver with led indication
- DS3231 RTC
- SWIM debug interface
